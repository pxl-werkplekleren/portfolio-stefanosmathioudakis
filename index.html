Stefanos Mathioudakis - Graduaad in programmeren

1. Inleidende tekst

Mijn naam is Stefanos en woon in Genk. Ik volg de opleiding graduaad in programmeren. Mijn interesse ligt vooral in de IT Sector, met een focus op het bouwen van websites. Dit is een richting waar ik mij goed voel en mezelf verder wil ontwikkelen.

Tijdens mijn opleiding leer ik programmeren op C#. Daarnaast leer ik bij Data essentials hoe ik data moet opsporen en analyseren. Voor Web Essentials werk ik met HTML CSS in webstorm, waarbij ik leer werken met positionering en formulieren.

In mijn vrije tijd spendeer ik veel tijd met mijn familie en met mijn vriendin en ik game ook in mijn vrije tijd. Daarnaast doe ik ook vrijwilligerswerk bij de vrije radio met mijn 3 programma's. Vanuit deze ervaring wil ik in de toekomst ook helpen met het bouwen en verbeteren van websites
